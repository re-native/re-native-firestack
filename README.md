# re-native-firestack

A Clojurescript library for [react-native-firestack](https://github.com/fullstackreact/react-native-firestack)
Meant for use with reagent and re-frame

Tracking      | Artifact
--------------|---------|
`2.3.8`       | `[re-native/firestack "0.0.1-SNAPSHOT"]`

In your re-natal project:

```
npm i react-native-firestack@2.3.8 --save
re-natal use-component react-native-firestack
```

Then follow the instructions on the react-native-firestack github page
